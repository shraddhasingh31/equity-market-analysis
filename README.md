# Equity Market Performance & Volatility Analysis

## Objective
To analyze the return and risk characteristics of selected Indian stocks and identify stocks with better risk-adjusted performance.

## Dataset
- Source: Yahoo Finance
- Stocks analyzed: TCS, Reliance, HDFC Bank, Infosys, ITC
- Time period: 2019â€“2024

## Tools Used
- Microsoft Excel

## Analysis Performed
- Data cleaning and organization
- Calculation of daily and annual returns
- Volatility measurement using standard deviation
- Risk-adjusted return comparison

## Key Insights
- Some stocks delivered higher returns with significantly higher volatility
- FMCG stocks showed more stable performance compared to IT stocks
- Risk-adjusted returns favored stocks with moderate volatility

## Conclusion
The analysis indicates that stocks with balanced return and risk profiles are better suited for long-term investors.
